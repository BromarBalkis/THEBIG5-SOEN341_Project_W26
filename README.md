-MealMajor is a student-focused web app that helps users plan weekly meals, manage groceries, and discover or create simple recipes.
-It includes three core modules:

1)User Accounts

-Sign up / log in
-Manage a profile with dietary preferences and allergies

2)Recipes

-Create, edit, delete recipes (with details like ingredients, prep time, steps, and cost)
-Search and filter recipes by time, difficulty, cost, and dietary tags

3)Weekly Meal Planner

-Build a weekly plan displayed in a grid
-Assign recipes by day and meal type (breakfast/lunch/dinner/snack)
-Edit/remove planned meals
-Avoid duplicate recipes within the same week
